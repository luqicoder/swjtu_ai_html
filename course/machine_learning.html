机器学习

当前位置: 首页 >> 实验课程 >> 机器学习

机器学习实验课程是与机器学习B这门课程的配套实验课程，主要使用相关编程环境进行数据预处理、回归、聚类、分类等实验实现。同时完成一个课程设计的实验，采用多个同类算法，对多个数据进行实验，采用交叉验证，完成模型选择。


课程教学目标

  1. 掌握文献调研方法，掌握问题分析方法，能够对机器学习问题进行理解，并且对问题进行建模。形成问题的解决方案。
  2. 能够使用一种或者多种编程语言和编程环境实验各算法，通过编程实现理解算法相互之间的差别。能够进行对比研究与实验。
  3. 理解与掌握机器学习软件开发的编程语言，及使用集成开发环境，掌握机器学习软件项目管理和开发的一般流程。


课程内容

  1. 数据预处理：
    • 一般数据预处理
    • 降维
    • 特征提取
  2. 回归方法：
    • 完成线性回归实验
    • 尝试非线性回归实验
  3. 分类理论和算法：
    • K-近邻
    • 决策树分类方法
    • 特贝叶斯分类
    • 神经网络
  4. 聚类算法：
    • K均值算法
    • K代表点
    • 模糊均值算法
    • 高斯混合模型
  5. 集成学习：
    • Boosting算法
    • Adaboosting
    • 聚类集成算法
  6. 机器学习问题解决：
    • 完成一个辅助诊断系统的设计与实现
